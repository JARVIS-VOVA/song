# Song

## Example project that demonstrates basic CRUD operations with Node.js and Firebase

## Quick start

1. **Clone**
  ```bash
  git clone git@github.com:JARVIS-VOVA/song.git && cd song
  ```

2. **Create .env**
  ```bash
  cp .env.example .env
  ```

3. **Install dependencies**
  ```bash
  yarn
  ```

4. **Run server**
  ```
  yarn start
  ```
